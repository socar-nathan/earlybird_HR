- 개인의 성과가 업무의 성장과 비례하는가?, 라는 의문에서 시작하여
공무원 공직생활에 대한 인식조사 설문지를 바탕으로 성과지표를 데이터로 분석하여 상관관계를 입증하는 프로젝트를 진행한다.
